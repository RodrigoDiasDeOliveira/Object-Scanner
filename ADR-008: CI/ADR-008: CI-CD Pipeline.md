ADR-008: CI/CD Pipeline
Status: Accepted
Decision: Use GitHub Actions for automated testing and building of both backend and Android app.
Rationale: Ensure reliability of the data entry pipeline that feeds TLP.
Alternatives Considered: Other CI tools.
Pros:

Fast feedback
Consistent builds

Cons:

Limited free tier resources for heavy Android builds

Consequences: High confidence in releases that integrate with TLP.
